# Goal Tracker App

## My First Project
This is my first web development project.

## Description
Goal Tracker App is a simple web application that helps users track their daily goals and monitor their progress.  
Users can add goals, mark them as completed, and view their completion progress.

## Live Demo
https://goal-tracker-by-Swaroopa2026.netlify.app

## Features
- Add daily goals
- Mark goals as completed
- Track progress using a progress bar
- Delete goals
- Data stored using localStorage
- Simple and clean user interface

## Technologies Used
- HTML
- CSS
- JavaScript
